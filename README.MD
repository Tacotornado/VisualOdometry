# Installation Instructions #

run "pip install -r requirements.txt" to install dependancies

the main file we are running is visual_odo.py however we have other tests using orb_odometry.py

you can shange the source in the main program of the visual_odo.py file, please refer to comments in the code

to begin please calibrate the drone camera intrinsics using a checkerboard pattern and the cam_calibrate file.
